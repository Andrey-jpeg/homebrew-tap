# Andrey-jpeg Tap

## How do I install these formulae?

`brew install andrey-jpeg/tap/<formula>`

Or `brew tap andrey-jpeg/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "andrey-jpeg/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
